In this example we just load typescript.js as it is, creating a Program fails because it tries to access the native filesystem (ts.sys is undefined in the browser). So we need to provide a custom CompilerHost and LanguageServiceHost and that's what this project is all about.

TypeScript Program:

log = function (s) { document.getElementById('output').innerText += s + '\n' }
const program = ts.createProgram(['test1.ts'], {})
const sourceFile = ts.getSourceFile('test1.ts')
log(sourceFile.getText())
